A RESTful API which provides the backend for NC Games, utilizing a Model-View-Controller design pattern.

DogTags Native Mobile App using Tensorflow.js

A React Native mobile app using machine learning to classify photos of pet dogs by breed.

We used a process called 'transfer learning' to customise an existing general purpose image classification model for this specific use case.
