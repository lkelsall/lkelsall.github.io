NC Games ReactJS Web Frontend

A website for viewing, upvoting and commenting on board game reviews.

I styled the site using styled-components, a CSS-in-JS library for React.

You can view it live, and find the codebase on GitHub.

NC Games API Express.js Backend

A RESTful API which provides the backend for NC Games, utilizing a Model-View-Controller design pattern.

The user, review and comment data is stored in a PostgreSQL database.

You can view it live, and find the codebase on GitHub.

DogTags Native Mobile App using Tensorflow.js

A React Native mobile app using machine learning to classify photos of pet dogs by breed.

Developed in a small team using agile-inspired practices such as daily standups and pair programming.

More information can be found on GitHub.
